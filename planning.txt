# 🚆 BGS Sidoarjo Landing Page — Development Plan (Next.js)

## 🎯 Objective

Membangun **landing page modern minimalist (light mode)** untuk:

* Branding komunitas **BGS Sidoarjo**
* Menarik member baru
* Menampilkan kredibilitas (diakui PT KAI & aktivitas sosial)

---

## 🧱 Tech Stack

* Framework: Next.js (App Router)
* Styling: Tailwind CSS
* Deployment: Vercel
* Asset: Static (image/vector, no backend yet)

## 🎨 Styling Guidelines

* Background: putih (#FFFFFF)
* Text: hitam / abu gelap
* Accent:

  * kuning (CTA / highlight)
  * merah (secondary highlight)
* Banyak whitespace
* Gunakan Tailwind utility classes
* Hindari shadow berat (minimalist)

---

## ⚡ Development Steps (Step-by-Step)

1. Setup project:

   * `npx create-next-app@latest`
   * install Tailwind

2. Setup layout & global styles

3. Build sections satu per satu:

   * Hero → About → Activities → dst

4. Import semua ke `page.js`

5. Add responsiveness

6. Final polish:

   * spacing
   * typography
   * hover effects

7. Deploy ke Vercel
